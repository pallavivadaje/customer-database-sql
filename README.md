# Customer Database SQL Project

## 📌 Overview
This project demonstrates the implementation of a Customer Database Management System using SQL. It includes database creation, table design, sample data insertion, and analytical SQL queries to generate meaningful business insights.

## 🚀 Features
- Create a customer database
- Manage customer information
- Store product details
- Record sales transactions
- Perform SQL analysis using JOIN, GROUP BY, HAVING, ORDER BY, and aggregate functions

## 🛠️ Database Schema

The project consists of three tables:

### 1. Customer
- Customer ID
- Customer Name
- City

### 2. Products
- Product ID
- Product Name
- Category
- Price

### 3. Sale
- Sale ID
- Customer ID
- Product ID
- Quantity
- Sale Date

## 📊 SQL Queries Included

This project contains SQL queries to analyze business data, including:

- Total sales amount per customer
- Product with the highest units sold
- Total revenue per product
- Average quantity sold per category
- Customers who purchased more than once
- City-wise total revenue
- Top 3 customers by spending
- Most profitable product category
- Total products sold in March 2024
- Daily revenue report

## ▶️ How to Run

1. Open MySQL Workbench or any MySQL client.
2. Import or open the `Customer database.sql` file.
3. Execute the script.
4. The database, tables, sample data, and SQL queries will be created automatically.
